# Prompt Evaluation Framework

## Criteria

| Metric | Description |
|------|-------------|
| Accuracy | Output correctness |
| Completeness | Covers all required elements |
| Consistency | Repeatability of output |
| Safety | No sensitive data leakage |

## Method

1. Define test inputs
2. Run prompts across models
3. Compare outputs
4. Score results

## Output Template

- Pass/Fail
- Observations
- Improvements required
